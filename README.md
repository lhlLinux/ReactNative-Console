## Console

Écrit en React-Native, ce projet, effectué dans un cadre scolaire, est une interface à deux vues (via stack navigation) d'une console de contrôle pour un système de maison intelligente, permettant d'envoyer des commandes via MQTT à celle-ci, par exemple pour allumer ou fermer des lumières, activer ou désactiver le système d'alarme. La console demeure à l'écoute de la confirmation des commandes du système domotique pour ensuite les afficher à la console.

#### Technologies utilisées:
- Stack navigation : pour les vues
- MQTT : pour la communication à distance
- Async storage : pour stocker les commandes
